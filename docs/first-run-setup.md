# Privacy-Chromium Premium Edition - First Run Setup

## Welcome to Privacy-Chromium

On first launch, you'll be guided through a privacy-first setup experience.

## Setup Steps

### Step 1: Privacy Configuration

**Recommended Settings** (selected by default):
- ☑ Block third-party cookies
- ☑ Enable HTTPS-only mode
- ☑ Enable DNS-over-HTTPS
- ☑ Disable telemetry
- ☑ Enable tracking protection
- ☐ Allow Chromium to send diagnostic data (unchecked for privacy)

### Step 2: Search Engine Selection

**Available Options**:
- **SearXNG** (Recommended) - Privacy-respecting meta-search engine
- DuckDuckGo - No tracking search
- Startpage - European privacy search
- Qwant - French privacy search
- Ecosia - Environmentally-friendly search
- Swisscows - Swiss privacy search
- Mojeek - Independent search
- Custom - Add your own

### Step 3: Extension Selection

**Recommended Extensions**:

| Extension | Purpose | Privacy Impact |
|-----------|---------|----------------|
| uBlock Origin | Ad & Tracker Blocking | HIGH |
| Bitdefender TrafficLight | Phishing Protection | MEDIUM |
| Malwarebytes Browser Guard | Malware Protection | MEDIUM |
| Privacy Badger | Anti-Tracker | HIGH |

**Installation Options**:
- ☑ Install recommended extensions (Recommended)
- ☐ Skip extension installation

## Post-Setup Privacy Features

### Always-On Protections

These features are always active and cannot be disabled:

1. **No Google Services**
   - All Google connections blocked
   - Domain substitution active

2. **Zero Telemetry**
   - No usage statistics
   - No crash reports
   - No field trials

3. **Enhanced Isolation**
   - Storage partitioning
   - Cache partitioning
   - Origin isolation

## Troubleshooting First Run

### Issue: Extensions not installing

**Solution**:
1. Check internet connection
2. Verify Chrome Web Store access
3. Try manual installation from store
4. Restart browser and retry

### Issue: Privacy settings not applying

**Solution**:
1. Restart browser
2. Clear browsing data
3. Reset settings to default
4. Reconfigure privacy options

### Issue: Search engine not working

**Solution**:
1. Verify search URL format
2. Check HTTPS availability
3. Try alternative privacy engine
4. Report issue if persistent

## Data Privacy Notice

Privacy-Chromium Premium Edition:

1. **Does NOT collect**: Usage data, browsing history, personal info
2. **Does NOT send**: Telemetry, crash reports, crash dumps
3. **Does NOT track**: User behavior, search queries, sites visited
4. **Does store locally**: Bookmarks, saved passwords (encrypted), preferences

All data remains on your device unless explicitly shared.
